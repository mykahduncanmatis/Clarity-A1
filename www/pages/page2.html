Environment

  • Enclosure Size

    The Minimum size of a dart frogs enclosure should be 12" x 12" x 12". It is recommended to have more height than width.

  • What is a BIOACTIVE enclosure?
    Bioactive enclosures are the key to keeping dart frogs. There are many different elements that go into setting up Bioactive terrariums.
  • Humidity and Temperature
    Humidity: 70% to 100% according to Jungle Jewel Exotics.
